How to create Gmail account?

  • To create a Gmail account, first visit the Gmail website.
  • Click on the "Create account" button, which is located below the sign-in form.
  • Fill in your personal information, including your first and last name, desired email address, and password.
  • Follow the prompts to enter additional details such as your phone number (for verification purposes), recovery email, date of birth, and gender.
  • Review and accept Google’s Terms of Service and Privacy Policy to complete the account creation process.

How to create Github account?

  • To create a GitHub account, go to the GitHub website.
  • Click on the "Sign up" button located in the top right corner of the page.
  • Fill in the registration form with your desired username, email address, and password.
  • Complete the account verification process, which may involve solving a puzzle or receiving a verification email.
  • Optionally, follow the prompts to choose your plan and personalize your account settings.

How to connect VS Code Editor to GitHub Account?

  • Ensure that Git is installed on your computer by downloading it from git-scm.com and following the installation instructions.
  • Open Visual Studio Code.
  • Go to the Extensions view by pressing (⇧⌘X / Ctrl+Shift+X), search for "GitHub", and install the GitHub extension.
  • Click on the Accounts icon in the Activity Bar and sign in with your GitHub credentials.
  • Open the terminal in VS Code by pressing (Ctrl+`) and configure your Git settings by entering the commands.

How to clone/pull repository from GitHub?

  • Open Visual Studio Code. To clone a repository, go to View > Command Palette by pressing (Ctrl+Shift+P)
  • Type Git: Clone and select it from the list of options.
  • Enter the URL of the repository you want to clone (e.g., https://github.com/username/repository.git).
  • Choose a directory on your computer where you want to clone the repository.
  • To pull changes from the repository, open the Command Palette again by pressing (⇧⌘P / Ctrl+Shift+P).
  • Type Git: Pull and select it to pull the latest changes from the remote repository.

How to publish the repository as live site?

  • Ensure that your repository has an index.html file, which will serve as the homepage of your live site.
  • Go to the repository on GitHub and click on the "Settings" tab.
  • Scroll down to the "GitHub Pages" section.
  • Under "Source", select the branch you want to publish, such as main or gh-pages.
  • Choose the folder from which to serve your site, such as /root or /docs. Save the changes.
  • Your site will be published at https://username.github.io/repository, where username is your GitHub username and repository is the name of your repository.
